## Overview

This project demonstrates an end-to-end Data Analytics workflow using Python, SQL, and Power BI. The objective is to analyze a dataset, clean and transform the data, perform exploratory data analysis (EDA), execute SQL queries for business insights, and build an interactive dashboard for visualization and reporting.

The project also includes a professional report and presentation created using Gamma for stakeholder communication.
